=== Contact Form 7 Proxy IP ===
Requires: 4.7
Tested up to: 4.7.3
Stable tag: 0.1.0
Contributors: Rudloff

== Description ==
This WordPress plugin adds a new [_forwarded_ip] tag to Contact Form 7's e-mail templates. This tag will contain the forwarded IP address if your website if behind a proxy.
